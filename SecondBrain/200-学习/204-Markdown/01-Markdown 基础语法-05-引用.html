引用语法

单个引用

> 引用内容

> 和引用内容之间有空格（space）

多个段落的块引用

> 段落1
>
> 段落2

段落之间的空白行添加一个 > 符号

嵌套块引用

> 引用内容1
>
>> 嵌套引用

在要嵌套的段落前添加一个 >> 符号

带有其它元素的块引用

引用内容里面可以添加其他 Markdown 语法格式
